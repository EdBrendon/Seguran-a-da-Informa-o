Hedy Lamarr:

Hedy Lamarr foi uma atriz e inventora austríaca que co-patentou um sistema de comunicação sem fio durante a Segunda Guerra Mundial, precursor da tecnologia de espectro expandido que é fundamental para as comunicações modernas, incluindo o Wi-Fi e o Bluetooth. Sua contribuição é muitas vezes subestimada, mas ela desempenhou um papel significativo no desenvolvimento de tecnologias essenciais para a comunicação sem fio.

Topologia:

Topologia de rede se refere à estrutura física ou lógica de uma rede de computadores, incluindo como os dispositivos estão conectados e como os dados são transmitidos entre eles. Existem diferentes tipos de topologias, como estrela, anel e malha, cada uma com suas próprias características e aplicações específicas.

WiFi:

Wi-Fi é uma tecnologia de comunicação sem fio que permite a conexão de dispositivos eletrônicos à internet e entre si, usando ondas de rádio. É uma das formas mais comuns de acesso à internet em todo o mundo, oferecendo conveniência e mobilidade aos usuários.

Comutador:

Um comutador, também conhecido como switch em inglês, é um dispositivo de rede usado para conectar dispositivos em uma rede local (LAN). Ele direciona o tráfego de dados apenas para os dispositivos destinatários, tornando as comunicações mais eficientes e seguras do que os hubs.

Blackbox:

Uma caixa-preta, ou blackbox, refere-se a um dispositivo ou sistema que registra informações e dados relevantes, geralmente em contextos de aviação ou investigação de acidentes. No contexto da tecnologia, pode se referir a qualquer sistema ou componente que opera de forma autônoma ou cujo funcionamento interno não é facilmente compreendido pelos usuários finais

Linguagem de programação:

As linguagens de programação, como Python, C# e JavaScript, são ferramentas essenciais para o desenvolvimento de software. Python é valorizado por sua simplicidade e versatilidade, enquanto C# é amplamente utilizado para aplicativos Windows e jogos, e JavaScript é fundamental para o desenvolvimento web. Cada uma tem suas próprias características e usos, adaptando-se a diversas necessidades de programação.

Pré-Projeto:

Um pré-projeto, muitas vezes referido como proposta de projeto, é uma etapa inicial de planejamento em diversos contextos, como pesquisa acadêmica ou desenvolvimento de projetos. Ele geralmente inclui uma descrição do problema a ser abordado, os objetivos do projeto, a metodologia a ser empregada e os recursos necessários para sua realização, fornecendo uma visão geral do trabalho planejado antes de sua execução.
